Veille technologique

Le référencement

Dans le cadre du BTS SIO, on nous demande d’effectuer une veille technologique sur un sujet de notre choix lié de préférence à notre spécialité, c’est pourquoi j’ai choisi le référencement, c’est-à-dire toute l’actualité concernant les méthodes pour référencer son site web ou son application mobile le mieux possible sur Internet en analysant et en prenant en compte les différentes mises à jour des algorithmes de référencement.

Pourquoi ce choix ? Tout simplement car il est selon moi étroitement lié à l’univers du développement. Quel intérêt de développer un site de qualité ou une superbe application mobile si personne n’y a accès ?

Il est important également de comprendre comment fonctionne le référencement, comment faire augmenter la visibilité d’un site web sur les moteurs de recherche (SEO : Search Engine Optimization). Pour cela je vous propose une infographie que je trouve particulièrement claire disponible sur le site axenet.fr qui reprend un à un les différents critères à prendre en compte pour réussir sa stratégie de référencement.

J’ai ensuite relevé quelques mises à jour importantes sur les algorithmes de Google concernant le référencement, cette liste n’est absolument pas exhaustive mais, elle permet de bien comprendre que les choses bougent rapidement dans ce milieu et qu’il est nécessaire de prendre en compte beaucoup de facteurs (qualité, rapidité, sécurité..) pour assurer à votre site ou votre application la meilleure place possible !
